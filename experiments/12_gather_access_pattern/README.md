# Experiment 12: Gather Access Pattern

## Overview
This folder contains experiment-local planning artifacts and implementation scaffolding.

## Files
- `experiment_plan.md`: experiment lecture-note plan used as the canonical technical spec.
- `architecture.md`: runtime architecture, shader layout, and measurement contract.
- `development_plan.md`: development sequencing and status tracking.
- `implementation_plan.md`: implementation work package checklist.
- `results.md`: run report with measurements, charts, and data links.
- `scripts/collect_run.py`: local helper that archives benchmark JSON into `runs/`.
- `scripts/analyze_gather_access_pattern.py`: builds summary tables from the latest run export.
- `scripts/plot_results.py`: renders charts from the generated tables.
